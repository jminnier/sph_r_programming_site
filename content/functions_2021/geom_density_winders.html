What is it for?

geom_density() is a function from the ggplot2 package that creates smooth density estimate plots, which are the smoothed versions of histograms. It is great for visualizing distribution, skewness, and kurtosis of continuous data. Many choose to overlay density plots over histograms with geom_density(), or create multiple density plots from different variables.

Is it helpful?

It is extremely helpful when performing exploratory data analysis to assess distribution of numerical variables and compare them to each other. Evaluating descriptive statistics of the data go hand-in-hand with visualization to get a better understanding of the story the data tell. This is why geom_density() is a powerful function.
